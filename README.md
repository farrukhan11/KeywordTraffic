# Keyword Traffic

Fresh Next.js application using MongoDB and Auth.js credentials authentication.

## Setup

1. Copy `.env.example` to `.env.local`.
2. Add `MONGODB_URI` and `AUTH_SECRET`.
3. Run `npm install`.
4. Run `npm run dev`.

## Current phase

- MongoDB user registration
- Auth.js / NextAuth credentials login
- Protected server-rendered dashboard
- Project creation
- Bulk keyword paste with normalization and duplicate removal
